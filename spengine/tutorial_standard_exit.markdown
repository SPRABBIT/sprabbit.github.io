##退出

退出程序的方法除了点击右上角的“X”外，还可以执行 @exit 命令。

@exit 的格式如下：

<pre class="brush: ks">
// 完整参数列表（无参数）
@exit;
</pre>

当程序执行到此命令，将直接退出程序。
