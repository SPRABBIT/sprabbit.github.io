部署 SpeShow

下载完毕后，解压压缩包，直接执行 SpeShow.exe 文件即可运行本程序。

开始运行后，SpeShow将做一些读取以及初始化工作，并执行 pre.ks 文件中的内容。pre.ks 是一个用KScript语言书写的脚本文件，KScript相关内容，将在下一节中描述。在预读资源并执行 pre.ks 的过程中会有读取动画播放（默认为一个转动的圆形）。读取完毕后，SpeShow 将执行在 pre.ks 中用 @first 函数指定的文件。如果没有在 pre.ks 中设置的话，默认会执行 first.ks 文件。

在执行之前，您还可以设置一些程序的初始化参数。使用记事本打开 config.cfg 文件，其内容大概如下:


    1024
    768
    800
    600
    0
    0
    0
    true
    true
    1

您可以更改其中的数值，程序运行时会根据这些数据初始化引擎。

参数说明如下：

属性名值类型说明
WindowWidth数字窗口宽度。
WindowHeight数字窗口高度。
WorkingWidth数字画布的宽度，编写脚本时以此为准。
WorkingHeight数字画布的高度，编写脚本时以此为准。
MultisampleType数字多重采样的类型，可以是0-8之间的数值，会影响图像显示质量。但若超过硬件能力，程序将运行失败。建议设置为0。
MultisampleLevel数字多重采样的级别，级别数值不得大于多重采样类型的数值。
RefreshRate数字屏幕刷新率，设置为0则为当前的屏幕刷新率。
Windowed布尔值窗口模式，若为true，则为窗口化显示，若为false，则为全屏显示。
VSync布尔值垂直同步，若为true则为打开，false则为关闭。
DeviceType数字DirectX设备类型，1为硬件设备，2为软件设备。建议为1。
